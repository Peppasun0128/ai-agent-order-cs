# AI Agent 訂單客服系統實戰 (Fine-tuned Qwen2.5)

本專案為 AI 新秀計畫第七期實作成果，建立了一個具備 **Function Calling** 能力的 AI 客服代理。

## 🚀 技術亮點
- **資料工程**：利用 vLLM 自動合成 200 筆高品質 SFT 訓練資料。
- **模型微調**：使用 **LoRA** 技術微調 Qwen2.5-3B-Instruct，顯著提升 JSON 輸出穩定性。
- **效能指標**：微調後模型在 32 項測試案例中達成 **100% 工具選擇準確率**。
- **後端開發**：使用 **FastAPI** 封裝模型 API，並實作 Trace 觀測與防呆重試機制。
- **容器化部署**：提供 Dockerfile 支援快速部署。

- **工具選擇準確率 (tool_acc)**: 100.0%
- **參數相符率 (args_exact)**: 100.0%
